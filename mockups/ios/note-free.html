PDF 问答助手

想做一个工具：上传一份 PDF，就能直接对它提问，不用自己把整份读完。

大概流程：上传后把文档切成小块 → 转成向量存起来 → 用户提问时先检索最相关的几段 → 把这几段连同问题一起丢给 Claude 回答。

后端先用 Supabase（自带 pgvector 向量扩展）。前端对话用 Vercel 的 AI SDK，流式输出体验好。

提示词最关键 👉 一定要让它「只根据检索到的内容回答，找不到就说不知道，绝对不要编」。

晚点看那个讲 RAG 的 YouTube 教程，还有 GitHub 上 langchain 的例子。
